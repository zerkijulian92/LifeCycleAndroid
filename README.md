Tentang Aplikasi
-------
LifeCycleAndroid  adalah aplikasi yang memperlihatkan bagaimana Lifecycle Activity di android studio itu berjalan


Aplikasi Ini Menggunakan
-----------
- Android Studio
- Java
- XML
